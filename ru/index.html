MATH PROBLEM GENERATOR APP

Математика прекрасна. Используя её, можно сделать множество вещей.
Изучай и практикуй математику, решая различные задачки, созданные в одно нажатие.
(Только для Windows)

Давайте начнём

Начать прекрасное путешествие в мир математики, скачав приложение.

Немного информации

о проекте

1. Интернет соединение для приложения

Интернет соединение не важно для приложения для правильной работы, но оно необходимо для обновления приложения

2. Немного о математических задачках

Приложение создаёт разные математические задачи, начиная с простых и заканчивая немного интересными. Эти задачи создаются по различным темам, число которых ограничено.

3. Обновления необязательны

Если доступна новая версия, приложение не будет обновляться без разрешения пользователя.

О нас

Короткая история:

Идея о создании генератора для задач по математике появилась случайно и была интересной. Возможно результат этой идеи кому-нибуть поможет в понимании довольно простой математики.
Извините за супер короткую историю, создавать более длинную сложновато.

Цели:

1. Упростить создания задач по матиматике.
2. Автоматизировать создание задач по математике.
